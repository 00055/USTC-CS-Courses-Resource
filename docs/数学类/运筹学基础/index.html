2018年

根据题目序号回忆, 题目分数 10分,12分,15分

  1. 填空题,10个空10分,有单纯形法的矩阵表示,图的支撑树, 对策值等
  2. 对偶理论, 给出最优解,求一个系数的值, 然后求对偶问题的解
  3. 第一问线性规划(没有要求方法,我用的图解法),第二问目标规划, 只需要列出模型
  4. 运输问题 要求用 Vogel 得到初解
  5. 0-1规划
  6. 指派问题,5-5
  7. 最大流,最小截
  8. 对策论, 第一问求 VG, 第二问变了一个赢得矩阵, 利用对策基本定理容易求相应的对策值与对策
